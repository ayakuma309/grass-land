# 🌱 GrassBadge - 「草を生やすたび、世界がちょっとだけ楽しくなる。」

[my-grass-garden-front.vercel.app](https://my-grass-garden-front.vercel.app)

## 🌱 サービス概要
GrassBadgeは、GitHubの活動（草）をもとに、自分自身の成長をゲーム感覚で楽しめる学習可視化アプリです。

他人と比較せず、「昨日の自分」に勝つことを応援します！

## ❤️ このサービスへの思い・作りたい理由
このサービスの出発点は、学習者のリアルなつまずきにあります。
- 「毎日続けたいのに、すぐ折れてしまう」
- 「他人の進捗に焦って、つらくなる」
- 「比べてばかりで、自分を認められない」

プログラミングを学び始めた多くの人がこのような悩みを抱えています。
私自身、周囲の学習者やスクール生から、そういった声をたくさん耳にしてきました。

だからこそ、「成果」ではなく「過程」を褒めることにフォーカスしたアプリを作りました。
笑いの要素・演出・称号などを加えることで、**“真面目に楽しく続けられる”** 場を提供したいという想いから生まれたサービスです。

## 📱 サービスの利用イメージ
1. GitHubアカウントでログイン
2. バッジ図鑑へ遷移しバッジ取得ボタンを押す
3. GitHubの貢献度(草)に応じて称号・バッジがもらえる

### 得られる価値
- 自分の進捗に喜べる体験
- 「やる気が出ない日」の再スタート支援
- 達成感と笑いが共存する学習サポート

## 📣 ユーザーの獲得について
- プログラミングスクール生へ向けて発信
- SNSでのシェア促進（バッジ・草画像のシェア機能）
- コラボ企画

## 💎 サービスの差別化ポイント・推しポイント
| 比較項目          | Findy                     | GrassBadge              |
| ------------- | ------------------------- | ---------------------- |
| 🎯 目的         | **エンジニア市場でのスキル証明・キャリア支援** | **学習継続のモチベ支援・習慣化サポート** |
| 👥 比較軸        | 他人との相対評価（スコア偏差）           | 昨日までの自分との絶対評価（勝敗）      |
| 🧠 コンセプト      | スキル評価型「ポートフォリオ」           | 自己肯定型「学習ガーデン」          |
| 🏅 バッジの意味     | スキルや実績の証明（就活用）            | ユーモア・応援・称賛（自分を褒める）     |
| 💬 表現         | シンプルでプロフェッショナル            | ネタ・癒し・ユーモアたっぷり         |
| 🎮 ゲーミフィケーション | 軽微（称号・数値）                 | 濃い（バッジ・演出・クエスト・ストーリー）  |
| 🧸 ターゲット      | 経験者・転職志望者向け               | 初学者・学習者向け（スクール生含む）     |
| 🌱 心理設計       | キャリア志向・市場評価軸              | モチベ維持・継続支援・比較回避設計      |

Findyは「他人に見せる」ためのスコアリング。GrassBadgeは「自分を励ます」ための草可視化。

### 推しポイント
- プログラミングスクール生特化
- 「草ホッピー同盟」や「講師バッジ」など、ユーモアで元気を届けるデザイン
- クエスト機能でタスクをこなすのではなく“冒険するように学ぶ”体験
- 成果を競わず、習慣化の支援に特化した設計
- 自分の努力が、**ちゃんと祝福される体験**を届けます。

#### 🏅 バッジ（全81種）
- レア度・称号文・取得条件あり
- コレクション感と笑いを両立

#### 🧭 クエスト（7カテゴリ × 各3〜5タスク）
- 「やることリスト」ではなく「自分の冒険」
- プレイヤーの成長曲線にあわせた難易度設計

#### 🔕 あえて“通知しない”設計
* 通知・リマインダー機能は**意図的に排除**
* 自分のタイミングで訪れたときに、ちょっと嬉しい発見がある

> 「やらされる」から「来たくなる」へ。
> 習慣化アプリの**理想の距離感**を目指しました。

#### 🌿 草の分類・評価方法にもこだわり
草にはいろんな生やし方がある。
だから、いろんなタイプの学習者が報われる設計を目指しました。

- コツコツ型、爆発型、週末集中型など、あらゆる傾向にバッジを用意
- 「継続のつらさ」に寄り添い、「自分なりの頑張り方」を肯定

## 機能候補
### 🚀 MVPリリース
- GitHub連携（GraphQL APIによる草・貢献データ取得）
- バッジ表示（取得条件と演出あり）(15~20種類)
- バッジ図鑑
  
### 🌿 第1フェーズアップデート
- 草の成長可視化（草の数、連続日数など）
- バッジ表示の拡充（取得条件追加・レア度演出強化）
- 昨日との比較機能（勝敗表示・進捗バー）
- マイページ機能
- 管理者機能
- 活動パターン分析のダッシュボード表示

### ✨ 第2フェーズアップデート
- 講師バッジ（限定実装）
- GitHub貢献カレンダー表示

### 🎉 本リリース
- クエストシステム（難易度別に小さなタスク群）
- SNSシェア機能(バッジ機能、クエスト機能)
- お問い合わせ画面

### 🌈 拡張予定
- マイページ編集
- 年間の活動記録
- クエスト進歩機能
- コラボ企画機能
- シークレット企画
