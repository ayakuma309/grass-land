# 🧬 Grass Type - 「あなたの草は、どんなタイプ？」

https://grass-type.vercel.app/

> GitHubの草が語る。あなたの“開発スタイル”を診断するアプリ。


## 🌱 サービス概要

Grass Typeは、GitHubの活動をもとに
あなたの **「草の性格」＝ 開発スタイル** を分析・可視化する診断アプリです。

単なるスコア表示ではなく、「あなたの努力をキャラクターとして表現」する体験を通して、
“自分を知るきっかけ”を届けます。

## ❤️ このサービスへの思い・作りたい理由
Grass Type の原点は、「学習者が自分を責めてしまう瞬間」にあります。

- 「バッジを取得できない」
- 「どこから頑張ればいいかわからない」
- 「頑張ってるのに成果が見えない」
そんな気持ちを抱いたことがある人に、
「あなたの努力には、ちゃんと個性がある」 と伝えたい。

「Grass Type」は、
「努力の量」ではなく「努力のスタイル」 に光を当てるために生まれました。

他人と比べる草じゃなく、
“あなたらしい草”を誇れる世界をつくりたい。

## 📱 サービスの利用イメージ
1. GitHubアカウントでログイン
2. 「診断する」ボタンを押す
3. GitHubの草データを取得してスコア化
4. あなたの草タイプが登場！（32タイプのうち1つ）
5. 結果をSNSでシェアしてみよう🌱

月に1回のペースで前回とキャラクターが変わったかどうかを確かめることができます！

### 🎁 得られる価値
- 自分の開発スタイルを客観視できる

- モチベが下がった時も、キャラクターでユーモアに
- 月ごとに診断して現在の自分を知れる

「あなたの草にも、ちゃんと個性がある。」
それを見つけるのが、Grass Typeです。

### 🧠 診断内容

GitHubデータをもとに、以下の5軸をスコア化し性格タイプを算出。

| 指標                  | 内容             | 意味するもの         |
| ------------------- | -------------- | -------------- |
| 🟩 継続力（Consistency） | 連続活動日数         | コツコツ型か？        |
| 🟦 集中力（Focus）       | 最大contribution数      | 集中して一気に進めるタイプ？ |
| 🟪 探究心（Curiosity）   | 使用言語数          | 新しい技術への挑戦度     |
| 🟥 協調性（Cooperation） | PR・Issue・レビュー数 | チーム貢献・対人協働度    |
| 🟧 成長度（Growth）      | 月ごとの伸び         | 昨日よりどれだけ成長したか  |

この5軸から **32タイプ（2⁵）** の草キャラクターを生成。

### 🌟 推しポイント
* **草タイプ32種 × レアキャラ3種**
  → 「自分の草がどんなキャラになるか？」でワクワクを演出。
* **スコアを数値で終わらせない**
  → 結果に「草キャラのひとこと」付き。
* **Grass Badgeとの自然連携**
  → Grass Typeで“知り”、Grass Badgeで“祝う”。
* **SNSでの拡散性を重視**
  → 診断結果カードを画像出力（OGP対応）。
* **GitHub公式APIによる正確なデータ分析**
  → 実際の活動を“やさしく”フィードバック。
